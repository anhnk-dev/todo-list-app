# README

* Ruby version: 3.0.4

* Rails version: 7.0.8

## Setting up the project
- Install Ruby 3.0.4 and Rails 7.0.8
- Run Bundle Install
```
bundle install
```
- Database creation
```
rails db:create
```
- Database migration
```
rails db:migrate
```
- Create seed data
```
rails db:seed
```
- Run local server
```
./bin/dev
```
- Login using account: `user@example.com / 123456`
